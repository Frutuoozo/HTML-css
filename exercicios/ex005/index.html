Lista em HTML

Lista ordenadas

Simplismente serve para fazer uma lista ordenada usando (ol) dentro colocar (li) como no exemplo abaixo:

  1. Começar a estudar HTML
  2. Começar a estudar CSS
  3. Começar a estudar JavaScript

Listas não ordenadas

Simplismete serve paar listas que nao presisão fazer sentido, usando a tag (ul) e denttro a tag (li), como no exemplo abaixo

  • html
  • css
  • java
  • javascript
  • progrmação

Lista de Definição

Simplismente uma lista que tera palavras e logo em seguida uma definição dessa palavra, usando as tags (bl), dentro (dd) 9 (dt), exemplos:

HTML
é o conteudo de uma pagina ou site
CSS
é o desing de uma pagina ou site
