Tipos de textos

Negrito/Destaque

Agora vamos escrever um texto em Negrito com a tag (strong).

Italico

Agora vamos ver um texto em Italico com a tag (em).

Marado/Marcação

Agora um texto a com marcação para da destaque a uma palavra de forma diferente com a tag (mark).

Texto grande/pequena

usando tags (big e small), como Grande e pequeno

Texo Excluido/Desconsiderado

usamos a tag (bel) para desconsiderar uma parte do texto.

Texto Inserido/Incluido

uamos a tag (ins) para inserir uma parte do texto.

Texto em cima

Usamos a tag (sup) para ter um texto em cima como: x9+b+c.

Texto em baixo

Usamos a tag (sub) para ter um texto em baixo como: H2O.
